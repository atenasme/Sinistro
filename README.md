
# PseudoC IDE

IDE web para ensinar lógica de programação usando uma linguagem pseudoC.

## Funcionalidades

- Editor de código
- Console integrado
- Entrada do usuário
- Variáveis
- Operações matemáticas
- IF / ELSE
- Execução no navegador
- Pronto para GitHub Pages

## Sintaxe

```txt
INICIO
GUARDAR x, y

LER(x)

SE(x > 10)
    MOSTRAR(x)
SENAO
    MOSTRAR(0)
FIMSE

FIM
```
