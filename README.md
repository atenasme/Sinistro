
# PseudoC IDE

IDE web para ensinar lógica de programação usando uma linguagem pseudoC.

## Funcionalidades

- Editor de código
- Console integrado
- Entrada do usuário
- Variáveis
- Operações matemáticas
- IF / ELSE
- Execução no navegador
- Pronto para GitHub Pages

## Sintaxe

```txt
INICIO
GUARDAR x, y

LER(x)

SE(x > 10)
    MOSTRAR(x)
SENAO
    MOSTRAR(0)
FIMSE

FIM
```

## Como publicar no GitHub Pages

1. Crie um repositório
2. Envie os arquivos
3. Vá em:
   Settings -> Pages
4. Escolha:
   Deploy from branch
5. Branch:
   main
6. Salve

Seu site ficará:

```txt
https://seuusuario.github.io/repositorio
```
